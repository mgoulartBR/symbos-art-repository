# Contributing (Artwork Submissions)

## How to submit
1. Fork this repository.
2. Add your files to the `/art/` folder:
   - `your_art.png` + `your_art.sgx`  OR  `your_art.jpg` + `your_art.sgx`
   - Both must share the same base filename.
3. Create a Pull Request to `main`.

## Rules
- Only PNG/JPG/JPEG + SGX are allowed.
- Files must be placed ONLY inside `/art/`.
- One image per SGX and vice-versa (no missing pairs).
- Do not upload ZIP/RAR, or any other file types.

## Recommended filename format
`title_author.png` + `title_author.sgx`
Example:
`kenshiro_zezinho2345.png`
`kenshiro_zezinho2345.sgx`
